Welcome to SPARTAN-Cloud's documentation!
===================================

**SPARTAN** (Surface PARTiculate mAtter Network) measures and provides surface ambient particulate matter (PM\ :sub:`2.5`\  and PM\ :sub:`10`\) concentration around the world. Check out `the SPARTAN website <https://www.spartan-network.org/>`_  for our motivation and collaborators worldwide!

This documentation will give examples of the data provided by SPARTAN and show how to use them. 
 
.. note::

   This project is under active development.

Welcome to SPARTAN-Cloud's documentation!
===================================

**SPARTAN** (Surface PARTiculate mAtter Network) measures and provides surface ambient particulate matter concentration around the world. Check out :RFC:`the SPARTAN website`  for our motivation and collaborators worldwide!

This documentation will give examples of the data provided by SPARTAN and show how to use them. 
 
.. note::

   This project is under active development.
